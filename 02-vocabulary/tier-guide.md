# 🧭 850 词分层学习指南（Cognitive Taxonomy Guide）

C.K. Ogden 建立的基本英语（Basic English, BE850）之所以能用 850 个词汇表达日常英语 90% 以上的概念，其核心秘密在于：**850 个词根绝非无序平铺的，而是经过精密认知设计、分层递进的网格体系。**

学习 BE850 时，**切忌使用传统的 A-Z 字母表死记硬背**。按照词汇在句子中的“杠杆率”（Leverage）与“认知维度”（Cognitive Dimension）进行分层学习，可使记忆效率与运用能力成倍提升。

## BE850 的五层认知分类法

Ogden 将 850 个词汇根据其在物理世界和语言结构中的角色，精细划分为以下五层：

```
                       ┌─────────────────────────┐
                       │   5. Opposites (50)     │  ←─ 对比记忆
                       ├─────────────────────────┤
                       │   4. Qualities (100)    │  ←─ 状态描述
                       ├─────────────────────────┤
                       │   3. General (400)      │  ←─ 名词化杠杆
                       ├─────────────────────────┤
                       │   2. Picturable (200)   │  ←─ 视觉映射
                       ├─────────────────────────┤
                       │   1. Operations (100)   │  ←─ 语法骨架与动词
                       └─────────────────────────┘
```

## 第一层：Operations 运作词 (100) — 核心骨架

*   **杠杆率**：★★★★★（最高）
*   **认知定位**：这是整个基本英语的**语法中枢和动力源**。它不仅包含了用于拼装一切动作的 18 个 Operator（如 be, do, go, put, take 等），还包含了所有代词（I, you）、方向介词（in, on, to）、限定词（this, that）、连接词（and, because）以及否定词（not）。
*   **学习要点**：
    *   **不要跳过**：这是最难的一层，但必须最先完全掌握。没有这 100 个词，你无法拼出任何一个合法的句子。
    *   **重点突破**：优先死磕 [18个 Operator 变位](../01-foundations/operators-18.md) 和 [方向词与空间几何](../01-foundations/directions-prepositions.md)。
*   **核心示例**：`come`, `get`, `give`, `go`, `keep`, `I`, `you`, `in`, `on`, `and`, `not`

---

## 第二层：Picturable 可画图物 (200) — 具体与形象

*   **杠杆率**：★★★★☆
*   **认知定位**：具体名词（人、动物、日常工具、身体部位）。这类词在大脑中最容易建立**直接实物映射**，不需要经过中文翻译的二次转换。
*   **学习要点**：
    *   **视觉记忆**：采用“看图认词”的方法。一个单词配一张图，在大脑中直接将英语声音/拼写与物理实物相绑定。
    *   **经典材料**：推荐配合 I.A. Richards 著名的 [*English Through Pictures*](../04-practice/reading-list.md)（画册第一册）进行情景记忆，该书即是以具体名词和 Operations 词逐步引入教学的。
*   **核心示例**：`dog` (狗), `hand` (手), `door` (门), `tree` (树), `eye` (眼睛), `house` (房子)

## 第三层：General Things 抽象一般物 (400) — 名词化替换

*   **杠杆率**：★★★☆☆（但数量最大）
*   **认知定位**：表示抽象概念、物理量、社会关系、活动状态的名词。这是 BE850 中体量最大的一块（占了接近一半的词汇）。
*   **核心理念（名词化替换）**：
    *   基本英语不使用复杂动词。当需要表达复杂的动作时，我们**将动作转化为对应的抽象名词**，再配合第一层的 Operator 组合出来。
    *   通过将“动词”转化为“运作动词 + 抽象名词”的形式，大幅降低了新动词词根的记忆量。

### 常用动词的名词化替换对照表：

| 想要表达的动作 (标准英语) | BE850 组合形式 (Operator + 抽象名词) | 直译物理感觉 |
| :--- | :--- | :--- |
| **decide** (决定) | **make a decision** | 做出一个决定 |
| **fear** (害怕) | **have a fear of** | 拥有一种对……的恐惧 |
| **discuss** (讨论) | **have a discussion** | 进行一场讨论 |
| **love** (爱) | **have love for / give love to** | 对……拥有爱 / 给予爱 |
| **shame** (羞辱) | **give shame to** | 给予羞耻感 |
| **state** (陈述/说明) | **make a statement** | 做出一个陈述 |
| **laugh** (笑) | **give a laugh** | 投以一笑 |
| **attack** (攻击) | **make an attack** | 发起一次攻击 |
| **argue** (争论) | **have an argument** | 拥有一场争吵 |

*   **核心示例**：`reason` (理由), `system` (系统), `idea` (主意), `fact` (事实), `way` (方法), `part` (部分), `act` (行为)

## 第四与第五层：Qualities & Opposites 性质与反义词 (100+50)

*   **杠杆率**：★★★☆☆
*   **认知定位**：性质词（Qualities）即形容词，用于修饰名词状态；反义词（Opposites）是其中 50 个性质词的直接相反词。
*   **学习要点**：
    *   **对比成对记忆**：不要将 Qualities 和 Opposites 分开背诵。将它们以成对（Antonym Pairs）的形式绑定记忆，可以缩短一半的脑部检索时间。
    *   *例*：背诵 `clean`（干净的）的同时，顺便背诵 `dirty`（脏的）。
    *   **衍生规则**：记住性质词可以通过加 `-ly` 变副词，加前缀 `un-` 变否定，以及前缀 `more/most` 变比较级（详见 [语法规则卡](../01-foundations/grammar-rules.md)）。
*   **核心示例**：
    *   `good / bad` (好/坏)
    *   `long / short` (长/短)
    *   `awake / asleep` (醒/睡)
    *   `true / false` (真/假)

## 📅 36天分层学习黄金日程表

基本英语官方推荐的学习计划是分为五个阶段，在 36 天内掌握全部 850 词。

| 天数阶段 | 阶段任务 | 每日目标 | 核心策略与配套材料 | 杠杆率 |
| :--- | :--- | :--- | :--- | :--- |
| **第 1–4 天** (4天) | **🔶 攻克 Operations 骨架** | 25 词 | 死磕 18个 Operator 变位和介词物理图解。完成站内“搭骨架”章节。 | ★★★★★ (最高) |
| **第 5–12 天** (8天) | **🟢 扫荡 Picturable 形象** | 25 词 | 配合站内 850 词详情页的视觉图片，阅读 *English Through Pictures (Book 1)* 课文。 | ★★★★☆ (最易) |
| **第 13–28 天** (16天) | **🔵 稳步推进 General 抽象** | 25 词 | 核心攻克名词化替换（把动词转换成 Noun 表达）。多读 reduction 改写句。 | ★★★☆☆ (最大量) |
| **第 29–33 天** (5天) | **🟣 记忆 Qualities 性质** | 20 词 | 熟悉性质词修饰名词的语法，学习 `-ly` 副词变形规则。 | ★★★☆☆ (搭修饰) |
| **第 34–36 天** (3天) | **🔴 成对掌握 Opposites 反义** | 17 词 | 将反义词与性质词放在一块，进行两极对比测试，查漏补缺。 | ★★☆☆☆ (成对背) |
