# Ogden's Basic English (BE850) 语法规则指南

Ogden 的基本英语（Basic English, BE850）在语法设计上秉承了一个核心理念：**语法与标准英语完全一致，但通过词汇的精简与结构的规范化，极大地减少了记忆负担。**

BE850 的核心在于**消除普通动词**，代之以 18 个“运作词”（Operators）与“方向词”（Directions）的组合来表达所有动作。因此，BE850 的语法规则专注于如何处理这 850 个词汇的变形、派生与句式构建。以下是 Ogden 官方制定的**十条核心语法规则**的详细讲解。

## 1. 复数变化规律（Plural: -S / -ES / -IES）

BE850 中的名词复数变化规则与标准英语完全一致：

### 1.1 基本规则
*   **一般情况**：在词尾加 **`-s`**。
    *   `dog` → `dogs`
    *   `cat` → `cats`
    *   `book` → `books`
*   **以咝音结尾（-s, -sh, -ch, -x, -o）**：在词尾加 **`-es`**。
    *   `glass` → `glasses` (玻璃杯)
    *   `brush` → `brushes` (刷子)
    *   `match` → `matches` (火柴)
    *   `box` → `boxes` (盒子)
    *   `potato` → `potatoes` (土豆)
*   **以辅音字母 + `y` 结尾**：变 `y` 为 `i` 再加 **`-es`**（即 **`-ies`**）。
    *   `story` → `stories` (故事)
    *   `baby` → `babies` (婴儿)
    *   `company` → `companies` (公司)
    *   `property` → `properties` (财产)
    *   `army` → `armies` (军队)

### 1.2 词尾辅音变化（F/FE → VES）
在 850 词表中，有几个以 `-f` 或 `-fe` 结尾的名词变复数时，需将 `-f/-fe` 变为 **`-ves`**：
*   `knife` → `knives` (小刀)
*   `leaf` → `leaves` (树叶)
*   `shelf` → `shelves` (架子)

### 1.3 词根不规则变化
850 词表中包含以下标准不规则复数名词，必须直接使用：
*   `man` → `men` (男人)
*   `woman` → `women` (女人)
*   `foot` → `feet` (脚)
*   `tooth` → `teeth` (牙齿)

## 2. 名词派生四种形式（Noun Derivation: -ER / -ING / -ED）

Ogden 规定，在 850 词表中的 600 个名词中，有 **300 个特定名词**（指动作或可产生动作的名词）允许添加后缀派生出四种衍生词，**且这些派生词不计入 850 词汇限额中**。

### 2.1 四种派生形式的定义

| 后缀 | 词性 | 语法意义 | 示例 (以 `work` 为例) | 中文释义 |
| :--- | :--- | :--- | :--- | :--- |
| **`-ER`** | 名词 | 动作的执行者或执行工具 | `worker` | 工人 / 工作者 |
| **`-ING`** | 名词 | 动作的过程、行为或具体产物 | `working` | 工作 / 劳动（行为） |
| **`-ING`** | 形容词 | 主动属性：正在进行该动作或具有该特征的 | `working (man)` | 工作着的（人） |
| **`-ED`** | 形容词 | 被动或完成属性：已被施加该动作的 | `worked (metal)` | 经过加工的（金属） |

### 2.2 派生拼写规则
*   **哑 `e` 结尾**：去掉 `e` 再加后缀。
    *   `love` → `lover`, `loving`, `loved`
    *   `judge` → `judger` (不常用，通常用 judge 本身表示法官), `judging`, `judged`
    *   `use` → `user`, `using`, `used`
    *   `hope` → `hoper`, `hoping`, `hoped`
*   **重读闭音节（单短元音 + 单辅音）结尾**：双写尾部辅音字母，再加后缀。
    *   `run` → `runner`, `running` (注：`run` 在 850 词中为名词词性，衍生出跑步者和跑步中)
    *   `stop` → `stopper` (塞子), `stopping`, `stopped`
    *   `drop` → `dropper` (滴管), `dropping`, `dropped`
    *   `slip` → `slipper` (拖鞋), `slipping`, `slipped`
*   **辅音 + `y` 结尾**：变 `y` 为 `i` 再加 `-ed`（`-ing` 保持不变）。
    *   `cry` → `crying`, `cried`
    *   `copy` → `copier`, `copying`, `copied`

## 3. 性质词变副词（Adverbs: Qualifier + -LY）

BE850 中的性质词（即形容词，Qualifiers）可以通过添加 **`-ly`** 后缀变为修饰动作方式的副词。

### 3.1 派生规律与拼写
*   **直接加 `-ly`**：
    *   `quick` (快的) → `quickly` (快地)
    *   `slow` (慢的) → `slowly` (慢地)
    *   `clear` (清晰的) → `clearly` (清晰地)
    *   `safe` (安全的) → `safely` (安全地)
*   **性质词以 `y` 结尾**：先变 `y` 为 `i`，再加 **`-ly`**。
    *   `happy` (快乐的) → `happily` (快乐地)
    *   `angry` (愤怒的) → `angrily` (愤怒地)
    *   `dirty` (脏的) → `dirtily` (脏兮兮地)
    *   `healthy` (健康的) → `healthily` (健康地)
*   **性质词以 `le` 结尾**：去掉 `e`，变 `e` 为 `y`（即变整个词尾为 **`-ly`**）。
    *   `simple` (简单的) → `simply` (简单地/仅仅)
    *   `feeble` (虚弱的) → `feebly` (虚弱地)
    *   `possible` (可能的) → `possibly` (可能地)

> [!NOTE]
> 850 词表中部分性质词如 `fast`、`hard` 等本身即可作为副词使用，无须重复添加 `-ly`。

## 4. 比较级与最高级（Comparison: MORE / MOST 与特殊用法）

为了简化表达，BE850 的标准规则是统一使用前缀 **`more`** 和 **`most`** 来表达性质词的程度变化，尽量避免规则繁杂的 `-er` 和 `-est` 词尾。

### 4.1 官方推荐结构
*   **比较级**：**`more` + Qualifier** (更……)
    *   `more complex` (更复杂)
    *   `more beautiful` (更美丽)
    *   `more sharp` (更锋利)
*   **最高级**：**`most` + Qualifier** (最……)
    *   `most complex` (最复杂)
    *   `most beautiful` (最美丽)
    *   `most sharp` (最锋利)

### 4.2 口语与书面习惯的例外
Ogden 指出，在实际交流中，学习者不可避免会遇到标准英语中的 `-er` 和 `-est` 词尾。因此，对于 850 词表里**单音节或部分双音节短词**，允许在习惯上使用 `-er` 和 `-est`：
*   `long` → `longer`, `longest`
*   `short` → `shorter`, `shortest`
*   `near` → `nearer`, `nearest`
*   `quick` → `quicker`, `quickest`
*   `hard` → `harder`, `hardest`
*   `wide` → `wider`, `widest`
*   `young` → `younger`, `youngest`
*   `old` → `older`, `oldest`

### 4.3 不规则比较级
对于基础词汇，保留标准英语中的不规则变化：
*   `good` → **`better`** → **`best`**
*   `bad` → **`worse`** → **`worst`**
*   `much / many` → **`more`** → **`most`**
*   `little` → **`less`** → **`least`**

## 5. 反义词与 UN- 前缀（Opposite: UN- 与反义词）

BE850 鼓励通过逻辑对比来减少词汇量。否定和相反概念有以下两种处理方式：

### 5.1 使用 UN- 前缀
在性质词（形容词）前加上前缀 **`un-`**，表达相反的属性：
*   `happy` (快乐) → `unhappy` (不快乐)
*   `clean` (干净) → `unclean` (不干净/肮脏)
*   `natural` (自然) → `unnatural` (非自然/做作)
*   `necessary` (必要) → `unnecessary` (不必要)
*   `wise` (聪明) → `unwise` (愚蠢)
*   `common` (常见) → `uncommon` (罕见)

### 5.2 优先使用词表自带的反义词
如果 850 词表中已经存在现成的成对反义词，应优先使用，而不是盲目加 `un-`：
*   `good` (好) / `bad` (坏)
*   `safe` (安全) / `poison` (毒 - 借指危险，但更标准的是 antonym 词组) 或使用 `dangerous` (不在词表中，用 `full of danger` 表达)。
*   `cold` (冷) / `warm` / `hot` (热 - `hot` 不在 850 中，用 `warm` 或 `very warm` 表达)。

## 6. 疑问与否定句式（Questions & Negatives: 倒装与 DO）

BE850 的句式变化完全遵循标准英语。

### 6.1 疑问句的构建
*   **系动词（be）与助动词/情态动词主谓倒装**：
    *   `You are happy.` → `Are you happy?`
    *   `He will come.` → `Will he come?`
    *   `I may go.` → `May I go?`
*   **普通运作词（如 see, take, give 等）使用助动词 `do / does / did` 引导**：
    *   `You see the dog.` → **`Do`** `you see the dog?`
    *   `He took the book.` → **`Did`** `he take the book?`

### 6.2 否定句的构建
*   **在系动词、助动词、情态动词后直接加 `not`**：
    *   `I am happy.` → `I am` **`not`** `happy.`
    *   `They will come.` → `They will` **`not`** `come.`
*   **普通动作句式使用 `do not / does not / did not` (或缩写 `don't / doesn't / didn't`)**：
    *   `I see it.` → `I` **`do not`** `see it.`
    *   `She made a mistake.` → `She` **`did not`** `make a mistake.`

## 7. 运作词与代词变位（Conjugation of Operators & Pronouns）

这是 BE850 语法中**唯一需要系统记忆变位**的部分。18 个 Operator 与代词拥有完整的标准英语人称、时态和语态变位。

## 8. 复合词组合规则（Compound Words）

为了不用背新单词就能表达新事物，BE850 允许由两个已有的 850 词拼接成一个复合词。

### 8.1 拼接类型
1.  **名词 + 名词** (Noun + Noun)：
    *   `milk` + `man` → `milkman` (送奶工)
    *   `rain` + `coat` → `raincoat` (雨衣)
    *   `sun` + `light` → `sunlight` (阳光)
    *   `coal` + `mine` → `coalmine` (煤矿)
    *   `school` + `room` → `schoolroom` (教室)
2.  **名词 + 方向词/介词** (Noun + Directive)：
    *   `sun` + `down` → `sundown` (日落)
    *   `day` + `light` → `daylight` (白昼/日光)
3.  **不定代词复合** (Pronoun Compounds)：
    *   `some` / `any` / `every` / `no` + `one` / `body` / `thing` / `where`
    *   `someone`, `somebody`, `something`, `somewhere`
    *   `anyone`, `anybody`, `anything`, `anywhere`
    *   `everyone`, `everybody`, `everything`, `everywhere`
    *   `no one`, `nobody`, `nothing`, `nowhere`

> [!IMPORTANT]
> 复合词的拼写应当是英文里已经广泛接受的拼写，不应随意捏造不符合英语习惯的词汇。

## 9. 度量衡、数字与日期（Measures, Numbers & Dates）

BE850 的词汇限制不包含数字、日期、标准度量单位等，这些内容直接使用标准英语的表达方式：

*   **数字（Numerals）**：`one, two, three, 100, 1000` 及其序数词 `first, second, third`。
*   **星期与月份（Calendar）**：`Monday, Tuesday...`, `January, February...`, `2026`。
*   **货币（Currency）**：`dollar, pound, cent, pence`。
*   **度量单位（Measures）**：`mile, foot, inch, yard`, `pound, ounce, gram, kilo`, `hour, minute, second`。

## 10. 国际通用词与专业术语（International & Technical Words）

为了让 BE850 更好地融入现实世界的专业场景，Ogden 在设计上为国际通用语和专业领域留出了开口：

### 10.1 国际通用词（约 50 个词）
这些词在世界绝大多数语言中都有相似的发音，因此不需要特意用 850 词去解释，可以直接使用：
`radio, hotel, taxi, telephone, telegram, television, cafe, bar, club, opera, beer, whisky, sport, tennis, golf, tobacco, cigarette, police, doctor, bank, gas, etc.`

### 10.2 专有名词（Proper Names）
人名、地名、国家名、节日名直接照常见拼写使用：
*   `George`, `London`, `China`, `America`, `Christmas`

### 10.3 行业专业词汇（Technical Jargon）
如果你在特定的职业（如医学、法律、工程、航空）中使用基本英语，允许引入该领域约 **100-150 个行业专属核心术语**，以便进行精准沟通。
*   例如在医学领域直接使用：`penicillin` (青霉素), `bacteria` (细菌) 等，不用 850 词拼写。

---

## 总结：如何判断一句话是否符合 BE850 语法？
1.  **词汇检查**：所有的实词（除数字、度量、专有名词、国际词外）是否都在 850 词表中？或者是基于这 300 个名词的合法派生词（`-er`, `-ing`, `-ed`）？
2.  **动词检查**：是否使用了 18 个 Operator 以外 of 普通动词（如 `read`, `write`, `decide`）？如果是，必须改写（例如：用 `do writing` 代替 `write`，用 `make a decision` 代替 `decide`）。
3.  **句法检查**：主谓变位是否正确？疑问和否定句是否使用了 `do` 倒装？代词的格是否正确？
