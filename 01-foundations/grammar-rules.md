# BE850 语法规则卡

> 对应 zbenglish [rules.html](https://zbenglish.net/sites/basic/rules.html) 十条规则的中文讲解。
> 例句保留英文——你学的就是这些句子怎么写。

Ogden 的核心说法：**语法和普通英语一样**，只是没有「普通动词」（靠 operator 拼动作）。
写出来听起来仍是正常、简单的英语。

## 十条规则

### 1. 复数加 -S

和正常英语相同：`dog` → `dogs`。

| 情况 | 规则 | 例 |
|------|------|-----|
| 一般 | 加 **-S** | `book` → `books` |
| 以 s/sh/ch/x 结尾 | 加 **-ES** | `glass` → `glasses` |
| 辅音 + y 结尾 | 变 y 为 **-IES** | `story` → `stories` |

### 2. 名词派生：-ER / -ING / -ED

约 300 个名词可以「长尾巴」，不增加新词根：

| 后缀 | 意思 | 例 |
|------|------|-----|
| **-ER** | 做这事的人/工具 | `work` → `worker`（工人） |
| **-ING** | 进行中 / 动名词 | `work` → `working` |
| **-ED** | 过去 / 被动 | `work` → `worked` |

### 3. 副词：性质词 + -LY

| 例 | 中文 |
|----|------|
| `quick` → `quickly` | 快地 |
| `slow` → `slowly` | 慢地 |

### 4. 比较级：用 MORE / MOST

BE850 **默认不用** `-er` / `-est`，用 more/most 更稳：

| 说法 | 例 |
|------|-----|
| 更… | `more complex`（更复杂） |
| 最… | `most complex`（最复杂） |

> 词表里已有 `better`、`more` 等词，口语里偶见 `-er`，但规则上优先 **MORE/MOST**。

### 5. 反义：UN- 或词表里的反义词

| 方式 | 例 |
|------|-----|
| 加 **UN-** | `happy` → `unhappy`（不快乐） |
| 用反义词 | `good` / `bad`，`big` / `little` |

### 6. 疑问句和否定：倒装 + DO

和普通英语一样：

| 类型 | 例 | 中文意思 |
|------|-----|----------|
| 疑问 | `Do you see it?` | 你看见了吗？ |
| 否定 | `I do not see it.` | 我看不见。 |

### 7. Operator 和代词：正常变位

16 个 operator + `be/do/have/say/see/seem` 按正常英语变化；
助动词只有 **`may`**（可以/也许）和 **`will`**（将/愿意）。

详见 [`operators-18.md`](./operators-18.md)。

### 8. 复合词

| 类型 | 例 | 说明 |
|------|-----|------|
| 两名词 | `milk` + `man` → `milkman` | 送奶工 |
| 名词 + 方向 | `sun` + `down` → `sundown` | 日落 |
| 不定代词 | `some` + `one` → `someone` | 某人 |

### 9. 数字、日期、度量

星期、月份、数字、单位——**按正常英语写**，不用 Basic 特殊规则。

### 10. 国际词和专业词

| 类型 | 例 | 说明 |
|------|-----|------|
| 国际通用词（约 50） | `radio`, `hotel`, `taxi` | 各国都认 |
| 当下任务需要的术语 | 医学、法律等 | 可用当地惯用说法 |
| 专有名词 | `London`, `George` | 照常见拼写 |

## BE850 故意不教什么

| 省略项 | 怎么办 |
|--------|--------|
| 普通动词（read, decide…） | 用 operator 拼：`make a decision` 代替 decide |
| 性数格 drill、所有格纠结 | 不搞；能省则省 |
| 动名词细分争论 | Ogden 认为对学习者无用 |
| -er/-est 比较级 | 用 more/most |

## 速查表

| 功能 | BE850 写法 | 例 |
|------|-----------|-----|
| 动作 | operator + 方向 | `put together`（组装） |
| 复数 | -S | `cats` |
| 人/工具 | 名词 + -ER | `worker`, `cutter` |
| 进行 | -ING | `working` |
| 过去 | -ED | `worked` |
| 副词 | 性质词 + -LY | `quickly` |
| 比较 | MORE / MOST | `more sharp` |
| 否定 | NOT / UN- / DO not | `not good`, `unkind` |
