# BE850 grammar rules (cheat sheet)

Ogden's claim: the rules are **identical to normal English** — minus ordinary verbs.
Output is "perfectly good, but simple, English." Below is the compact rule set
(the panel Ogden printed on the back of the word list).

## The rules

1. **Plurals** add **-S** (`dog ? dogs`; sibilants ? `-es` *glass?glasses*; `-y ? -ies` *story?stories*).
2. **Derived words** from ~300 nouns take **-ER, -ING, -ED**
   (`work ? worker / working / worked`; `-er` = agent or instrument).
3. **Adverbs** are made from qualities with **-LY** (`quick ? quickly`).
4. **Degree** uses **MORE / MOST**, not -er/-est (`more complex`, `most complex`).
5. **Opposites** often use **UN-** (`happy ? unhappy`) or a listed opposite word.
6. **Questions & negatives** use inversion and **DO** (`Do you see it?` / `I do not see it`).
7. **Operators** (16) + **pronouns** behave as in normal English; only **may / will** as modals.
8. **Compounds** are allowed: `any/every/no/some + one/thing/where`, `week-end`, `milkman`, etc.
9. **Numbers, days, months, measurement** in normal English.
10. **International words** (~50, e.g. *radio, hotel, taxi*) and **proper names** spelled as usual.

## What Basic English deliberately drops

- No ordinary verbs ? build with operators (`make a decision`, not "decide").
- No gendered grammar, no genitive drills, no gerund hair-splitting (Ogden's words).
- No -er/-est comparatives.

## Quick reference

| Function | BE850 way | Example |
|----------|-----------|---------|
| Action | operator (+ direction) | `put together` |
| Plural | -S | `cats` |
| Agent/tool | noun + -ER | `worker`, `cutter` |
| Ongoing | -ING | `working` |
| Past/participle | -ED | `worked` |
| Adverb | quality + -LY | `quickly` |
| Comparison | MORE / MOST | `more sharp`* |
| Negation | NOT / UN- / DO not | `not good`, `unkind` |

> *Some natural comparatives (`better`, `more`) are listed words; default pattern is MORE/MOST.
