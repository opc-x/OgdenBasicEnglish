# The 18 operators (all the "verbs" there are)

Basic English has **no ordinary verbs**. Action is carried by **16 operators** + **2 auxiliaries**.
Everything else ("read", "arrive", "assemble", "postpone") is built by **operator + direction/noun**.

## 16 operators

| Operator | Core sense | Typical compounds ? standard verb |
|----------|-----------|-----------------------------------|
| **come** | motion toward | come in (enter), come back (return), come out |
| **get**  | acquire/become | get up (rise), get off, get back, get out |
| **give** | transfer out | give up (surrender), give in (yield) |
| **go**   | motion away | go on (continue), go off, go out, go through |
| **keep** | maintain | keep on (continue), keep back, keep off |
| **let**  | permit | let in, let off, let out |
| **make** | create/cause | make up (invent), make out (perceive) |
| **put**  | place | put together (assemble), put off (postpone), put on, put out, put away |
| **seem** | appear | seem like |
| **take** | grasp/remove | take off (remove), take in, take up, take over, take away |
| **be**   | existence/state | be in, be off |
| **do**   | perform / aux | do again (repeat) |
| **have** | possess | have on |
| **say**  | utter | say again |
| **see**  | perceive | see through |
| **send** | cause to go | send up, send out, send off, send back |

## 2 auxiliaries (modals)

| Word | Use |
|------|-----|
| **may** | permission / possibility |
| **will** | future / volition |

## Why this works

A handful of operators æ ~20 directions æ nouns expresses thousands of specific actions:

```
take + off          ? remove
put  + together     ? assemble / build
make + up           ? invent
get  + up           ? rise
give + up           ? surrender / quit
put  + off          ? postpone
go   + on           ? continue
```

> Rule of thumb: if you want a verb that isn't an operator, **build it** from operator + direction,
> or rephrase with a noun (`make a decision` instead of "decide", `give an answer` instead of "answer-verb").
