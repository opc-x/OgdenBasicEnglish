# 词缀扩展：不增加新词根

Ogden 只允许固定几种词缀。**不算新词**，只是在已有词根上变形——850 根才能覆盖更多表面形式。

## 允许的词缀

| 词缀 | 加在什么上 | 变成什么 | 例 |
|------|-----------|----------|-----|
| **-S** | 名词 | 复数 | `dog` → `dogs` |
| **-ER** | 约 300 个名词 | 人/工具/比较 | `work` → `worker` |
| **-ING** | 同上 | 进行/动名词 | `work` → `working` |
| **-ED** | 同上 | 过去/分词/形容词 | `work` → `worked` |
| **-LY** | 性质词 | 副词 | `quick` → `quickly` |
| **UN-** | 性质词 | 反义 | `happy` → `unhappy` |

## 程度（不用 -er/-est）

| 形式 | 写法 | 例 |
|------|------|-----|
| 比较 | **MORE** + 性质词 | more complex（更复杂） |
| 最高 | **MOST** + 性质词 | most complex（最复杂） |

## 拼写小规则

- 咝音结尾复数加 **-es**：`glass` → `glasses`
- 辅音 + y：变 **-ies**：`story` → `stories`
- 哑 e + -ING/-ED：去 e：`make` → `making`

## 不允许

- 随便造 **-tion / -ment / -ize** 等新后缀 → 用词表里的名词代替
- 新动词词根 → 用 operator 拼
