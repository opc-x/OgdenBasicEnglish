# 短语动词：operator + 方向词

> BE850 的**发动机**。18 个 operator × 约 20 个方向 ≈ 替代约 4000 个英语动词。
> 需要动词时**拼出来**，别去背新动词。

## 另一条路：operator + 名词

拼不出短语动词时，用 **operator + 名词**：

| BE850 说法 | 普通动词 | 中文 |
|------------|----------|------|
| give an answer | answer | 回答 |
| make a decision | decide | 决定 |
| have a look | look | 看 |
| give attention | attend | 注意 |
| make a request | request | 请求 |
| do work | work | 工作 |
| have a desire | want | 想要 |
| make a comparison | compare | 比较 |
