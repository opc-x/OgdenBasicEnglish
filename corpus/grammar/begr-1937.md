# Basic English and Grammatical Reform (1937) — agent index

Full HTML mirror: `begr-1937.html`  
Source: https://zbenglish.net/sites/basic/begr.html  
Original pamphlet by C.K. Ogden; stated **no copyright** on publication.

## Core thesis

- English spelling and grammar are barriers; Basic English removes irregularities while keeping normal English syntax.
- 850 words + 18 operators replace ~25,000 dictionary entries for 90% of everyday concepts.
- Goal: international auxiliary language + faster path for non-native learners.

## Grammar rules (distill cards)

### Plural
- Add **-s** to nouns (regular plural).

### Comparison
- Use **more** / **most** instead of -er / -est (e.g. *more complex*, not *complexer*).

### Operators (verbs)
- Only 18 base verbs; complex meaning via compounds:
  - `put together` ? assemble
  - `make up` ? invent
  - `take away` ? remove

### Compounds
- Nouns combine: `birthday`, `football`, `workman`.
- Adjective + noun: `blackboard`, `goldfish`.

### Sentence structure
- Same as standard English (SVO); learner writes "perfectly good, but simple, English."

### Pronunciation
- Pamphlet includes phonetic reform notes; full detail in HTML sections II–IV.
